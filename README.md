# Task-3-Sales-Performance-Dashboard-Tableau
Task 3 - Interactive Business Dashboard using Tableau Public. Created a sales analytics dashboard with KPI metrics, category-wise sales, regional performance, profit analysis, sales trends, and interactive filters using the Sample Superstore dataset.

## Project Overview

This project was developed as part of the Data Analyst Internship Task 3. The objective was to design an interactive business dashboard that helps stakeholders analyze sales performance, profitability, regional trends, and category-wise performance using Tableau.

## Tools Used

* Tableau Public
* Microsoft Excel / CSV Dataset
* GitHub

## Dataset

* Sample Superstore Sales Dataset

## Dashboard Features

### KPI Cards

* Total Sales
* Total Profit
* Total Orders
* Growth KPI

### Visualizations

* Category Sales Analysis
* Regional Sales Analysis
* Profit Analysis by Sub-Category
* Sales Trend Analysis (Time Series)

### Interactive Features

* Category Filter (Slicer)
* Dynamic dashboard interactions

## Key Insights

* Technology category generated the highest sales performance.
* Regional analysis helps identify top-performing markets.
* Sales trend analysis highlights monthly performance patterns.
* Profit analysis reveals the most and least profitable sub-categories.

## Dashboard Objectives

* Monitor overall business performance.
* Track sales and profitability metrics.
* Analyze trends over time.
* Support data-driven decision-making.
